# ベビーカステラ 会計システム

## 背景

所属していたインカレバドミントンサークル（約60名）で学園祭への模擬店出店を企画した際、大学の「インカレ出店禁止」というルールが壁となりました。実行委員会が懸念していた会計の透明性を技術で担保するため、リアルタイムで売上を可視化できる会計システムを自ら開発しました。

## 機能

- 売上入力（プレーン・抹茶・チョコの3フレーバー対応）
- リアルタイム売上集計（合計・フレーバー別）
- 時間帯別売上集計
- 取引履歴の表示
- 誤入力時の削除機能
- ページをリロードしてもデータが消えない永続化

## 使用技術

- フロントエンド: HTML / CSS / JavaScript
- バックエンド: AWS Lambda（Python 3.12）
- データベース: AWS DynamoDB
- API: AWS API Gateway

## システム構成

ブラウザ (HTML/CSS/JavaScript)
↕
AWS API Gateway
↕
AWS Lambda (Python 3.12)
↕
AWS DynamoDB

## セットアップ

1. リポジトリをクローンする
2. `index.html` をLive Serverで開く
3. AWS環境を構築してAPI_URLを書き換える
